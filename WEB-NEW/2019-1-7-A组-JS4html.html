1.背景介绍

面向对象程序设计（object oriented Programming）作为一种新方法，其本质是以建立模型体现出来的抽象思维过程和面向对象的方法。摸具是用力反映现实世界中事务特征的。 任何一个摸具都不可能反映客观的一切具体特性，只能对事务特征和变化规律的一种抽象，且它所涉及的范围内更普遍，更集中，更深刻地描述客体特征。通过建立模型达到抽象是人们对客体的认识深化。

面向对象编程是一种编程思想，在JavaScript中，万物都是对象的基础上，面向对象编程的思想主要以对象为主，将一个问题抽象出具体的对象，并且将抽象出来的对象和对象的属性和方法封装成一个类。

知识剖析

面向对象编程与面向过程编程的区别

无论面向对象或者面向过程都是一种编程的思想，在面向对象和面向过程中，并不是完全相对的，也不是完全独立的。

面向过程编程主要是以动词为主，解决方法是按照顺序一步一步用不同的函数

面向对象是把构思问题分解各个对象，建立对象的目的不是为了完成一个步骤，而是为了描述某个事物在整个解决问题的步骤和行为。

上面创建了一个对象，并且创建了方法，在run函数种，this的指向很明显就是box，对象的本身。这是JavaScript创建对象的基本方法。缺点就在于，我们如果 要创建多个类似的对象的话，就会产生大量的代码。

为了解决多个类似对象的声明问题，我们可以使用一种叫做工厂模式的方法，这种方法就是为了解决实例化对象产生大量重复的问题。

function createObject(name, age) { //集中实例化的函数

var obj = new Object();

obj.name = name;

obj.age = age;

obj.run = function () {

return this.name + this.age + '运行中...';

};

return obj;

}

var box1 = createObject('Lee', 100); //第一个实例

var box2 = createObject('Jack', 200); //第二个实例

alert(box1.run());

alert(box2.run()); //保持独立

构造函数

1）函数名和实例化构造名相同且大写，(PS：非强制，但这么写有助于区分构造函数和 普通函数)；

2）通过构造函数创建对象，必须使用new 运算符。

任何函数，只要通过 new 操作符来调用，那它就可以作为构造函数 ； 　　任何函数，如果不通过 new 操作符来调用，那它跟普通函数也没有什么两样。

1）当使用了构造函数，并且new 构造函数()，那么就后台执行了new Object()； 2）将构造函数的作用域给新对象，(即new Object()创建出的对象)，而函数体内的this 就 代表new Object()出来的对象。 3）执行构造函数内的代码； 4）返回新对象(后台直接返回)。

语法： return[()[expression][]]; 可选项 expression 参数是要从函数返回的值。如果省略，则该函数不返回值。 用 return 语句来终止一个函数的执行，并返回 expression 的值。如果 expression 被省略， 或在函数内没有 return 语句被执行，则把值 undefined 赋给调用当前函数的表达式。
